# Tech Stack

## Programming Language
- C++

## Platform
- VEX V5 Robotics System

## SDK
- VEX V5 SDK (VEXcode)

## Build System
- Makefile (VEX Robotics VS Code Extension compatible)

## Architecture
- Modular competition structure with custom configuration isolation.
- Core logic resides in `src/` and `include/`.
- User-specific code and configuration reside in `custom/`.
